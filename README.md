# Mac Overflow

Menu bar overflow manager for macOS. Never lose your menu bar icons again!

## The Problem

macOS menu bar has limited space. When you have many apps with menu bar icons, some get hidden and become inaccessible - especially on MacBooks with notches.

**Native macOS doesn't handle overflow** - icons just disappear.

## The Solution

Mac Overflow adds a simple overflow dropdown to your menu bar. Click one icon (â‰¡) to see all your hidden menu bar items in a clean, organized list.
